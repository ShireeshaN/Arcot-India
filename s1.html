FEATURE

FEATURE 01

Creating spaces
by freely
controlling light

The brightness, color, lit area, and angle can all be controlled.
Using a timer relay, lighting can be automatically changed to preset scenarios.
We provide support for creating attractive spaces which expand the possibilities of presentation.

FEATURE 02

The lighting equipment is
operated from handy
smartphones or tablets

Using a wireless Bluetooth Mesh® connection,
up to 100 lights can be controlled from a single smartphone or tablet.
When a large number of lights need to be adjusted,
such as when switching displayed items or changing layouts,
being able to do so handily helps improve work efficiency and save human resources costs.
